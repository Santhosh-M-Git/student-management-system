# student-management-system

A comprehensive Student Management System developed using Spring Boot, Java, HTML, CSS, JavaScript, and MySQL. The system enables student, teacher, and admin management with attendance tracking, marks management, fee payments, leave requests, announcements, and study materials.

# 🎓 Student Management System

## 📌 Project Overview

Student Management System (SMS) is a web-based application designed to streamline academic administration and student-related activities. The system provides separate modules for Students, Teachers, and Administrators to efficiently manage attendance, marks, fee payments, study materials, announcements, and leave requests.

---

## 🚀 Features

### 👨‍🎓 Student Module
- Student Login
- View Attendance
- View Marks
- Access Study Materials
- Submit Leave Requests
- Receive Announcements and Alerts

### 👨‍🏫 Teacher Module
- Teacher Login
- Manage Student Attendance
- Update Marks
- Upload Study Materials
- View Student Information

### 👨‍💼 Admin Module
- Admin Login
- Manage Students
- Manage Teachers
- Dashboard Analytics
- Generate Reports
- Import & Export Data
- Manage Announcements

### 📊 Dashboard
- Total Students
- Total Teachers
- Attendance Statistics
- Marks Summary
- Fee Payment Status
